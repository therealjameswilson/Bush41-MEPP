Reproducibility

Ingest and Verification Trail

Record Harvest

node scripts/harvest-presidential-conversations.js queries the Presidential Memcon and Telcon Files, filters to unrestricted item-level PDFs, dedupes by NAID, and writes data/records.json.

Public Papers Harvest

node scripts/harvest-public-statements.js downloads official GovInfo Public Papers PDFs, extracts page-level text, promotes MEPP-relevant entries, and writes data/public-statements.json.

Supporting Data

node scripts/build-supporting-data.js rebuilds persons, event dossiers, source leads, and compiler gaps from the harvested corpora and curated compiler targets.

Source Anchors

Primary anchors: the FRUS Volume XIV page, NARA Catalog, GovInfo Public Papers, and George H.W. Bush Library declassified PDFs.
